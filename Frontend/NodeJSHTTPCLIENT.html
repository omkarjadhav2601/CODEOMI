REST Client

In Visual Studio Code you can install "REST Client" extension from the market to perform HTTP Requests. I have attached few screenshots of the example below for the same.

Points to Remember:

  • You can run the HTTP request in editor with the file extension as (.http) and then you will see the response in a seperate pane(Reslponse)
  • You can have mulitple requests in the same .http file seperated by a delimeter as three hash (###)

You can check for more details in VS Code or on the Official site.
